# Flash-Sale-Optimization-SQL-Case-Study
Analyzing 10,000+ e-commerce transactions to optimize product selection and pricing for Flash Sale campaigns using **SQL Server** and **Power BI**.

---

## 📑 Detailed Analysis Report
This project provides a data-driven framework for e-commerce promotional strategies, focusing on balancing volume growth with profitability.

👉 **[View Full Analysis Report (PDF)](https://github.com/BaoYing2403/Flash-Sale-Optimization-SQL-Case-Study/blob/main/Report.pdf)**

### 🛠 Key Tech Stack:
* **Database:** SQL Server (T-SQL) for Data Cleaning & Business Logic.
* **Analysis:** GMV, Profit Margin, Sales Lift & Velocity analysis.

---

## 🚀 Strategic Highlights (Summary)
* **Optimal Discount:** Identified a **10% discount** ceiling to maximize traffic without eroding margins.
* **Traffic Magnets:** **Kitchen and Toys** categories show the highest responsiveness with **~400% volume surge**.
* **Hidden Gems:** Targeted underperforming products with high margins (**25%**) for SEO growth.
